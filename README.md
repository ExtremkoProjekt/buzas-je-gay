# Creating of JAR in Netbeans
- Right-click on the Project name
- Select Properties
- Click Packaging
- Check Build JAR after Compiling
- Check Compress JAR File
- Click OK to accept changes
- Right-click on a Project name
- Select Build or Clean and Build
